# Star Seeker App

Explore the cosmos with ease using Star Seeker! My user-friendly app helps you plan interstellar journeys, calculate costs, and find the cheapest routes.
With sleek design and seamless functionality, it's your passport to the stars. 
Start your cosmic adventure now!

Created using React, with Redux as storage. 

Development ongoing!
